Peripleo operates on top of a JSON API ("Peripleo API v.2") that is open for public use. Work on proper documentation is underway.

Meanwhile, feel free to use our old Peripleo API ("Peripleo API v.1"), documented here:

https://github.com/pelagios/peripleo/blob/master/README.md

The API provides less functionality than the Peripleo API v.2. But it is served from the same infrastructure, and exposes the same data as the rest of Peripleo.
